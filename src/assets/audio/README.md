# 音频文件放置说明

## 📁 目录结构

```
VisionFocusHours/src/assets/audio/
├── background/              # 背景音乐
│   └── meditation.mp3       # 放置您的冥想背景音乐
│
└── voice/                   # 引导语音
    ├── section1.mp3         # 第1阶段语音
    ├── section2.mp3         # 第2阶段语音
    ├── section3.mp3         # 第3阶段语音
    ├── section4.mp3         # 第4阶段语音
    └── section5.mp3         # 第5阶段语音
```

## 🎵 文件说明

### 背景音乐
- **文件名**: `meditation.mp3`
- **位置**: `background/meditation.mp3`
- **时长**: 5-10分钟 (会循环播放)
- **音量**: 系统自动设置为30%

### 引导语音
- **文件名**: `section1.mp3` ~ `section5.mp3`
- **位置**: `voice/` 目录下
- **音量**: 系统自动设置为70%

## ✅ 系统行为

1. **如果音频文件存在** → 自动播放
2. **如果音频文件不存在** → 仅显示文字引导

## 🎯 自动播放逻辑

### 有语音文件时:
```
点击"开始冥想引导"
    ↓
播放背景音乐 (循环)
    ↓
播放第1阶段语音 + 显示全部文字
    ↓
语音播放完毕 (自动检测)
    ↓
等待3秒
    ↓
自动切换到第2阶段
    ↓
... 循环直到第5阶段完成
    ↓
显示"冥想完成"界面
```

### 无语音文件时:
```
点击"开始冥想引导"
    ↓
播放背景音乐 (循环)
    ↓
文字逐行显示 (按预设时间)
    ↓
当前阶段文字播放完毕
    ↓
等待3秒
    ↓
自动切换到下一阶段
    ↓
... 直到完成
```

## 🎙️ 推荐的录音参数

- **格式**: MP3
- **比特率**: 128kbps - 320kbps
- **采样率**: 44.1kHz
- **声道**: 单声道或立体声

## 📌 注意事项

1. 文件名必须完全匹配 (区分大小写)
2. 文件格式必须是 `.mp3`
3. 确保文件已上传到正确的目录
4. 音频文件会被浏览器缓存,更新后可能需要强制刷新 (Ctrl+F5)

## 🔧 测试方法

访问: `http://localhost:3000/pages/meditation.html?year=2026`

1. 打开浏览器开发者工具 (F12)
2. 切换到 Network 标签
3. 筛选 "Media" 类型
4. 点击"开始冥想引导"
5. 观察是否加载音频文件

### 成功: 
- 看到 `meditation.mp3` 和 `section1.mp3` 等文件加载
- 状态码: 200 (成功) 或 206 (部分内容)

### 失败:
- 状态码: 404 (文件不存在)
- 系统会自动回退到文字引导模式
