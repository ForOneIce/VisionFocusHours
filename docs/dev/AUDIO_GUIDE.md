# VisionFocus Hours - 音频文件说明

## 📁 音频文件结构

```
VisionFocusHours/src/assets/audio/
├── background/              # 背景音乐
│   └── meditation.mp3       # 冥想背景音乐 (循环播放)
│
└── voice/                   # 引导语音
    ├── section1.mp3         # 第1阶段: 冥想准备与开场 (~90秒)
    ├── section2.mp3         # 第2阶段: 呼吸锚定与身体放松 (~100秒)
    ├── section3.mp3         # 第3阶段: 自然意象与内心平静 (~90秒)
    ├── section4.mp3         # 第4阶段: 愿景设想的心理准备 (~80秒)
    └── section5.mp3         # 第5阶段: 结束与回归 (~60秒)
```

---

## 🎵 音频文件要求

### 背景音乐 (`meditation.mp3`)
- **格式**: MP3
- **时长**: 建议 5-10 分钟 (会循环播放)
- **风格**: 轻柔、舒缓、环境音乐
- **音量**: 预设为 30%
- **推荐**: 
  - 自然白噪音 (雨声、海浪、森林)
  - 轻柔钢琴/古筝
  - 冥想音乐/颂钵

### 引导语音 (`section1.mp3` ~ `section5.mp3`)
- **格式**: MP3
- **音质**: 清晰、温暖女声
- **语速**: 缓慢、平静
- **音量**: 预设为 70%

#### 各阶段时长:
1. **section1.mp3** - 冥想准备与开场: ~90秒
2. **section2.mp3** - 呼吸锚定与身体放松: ~100秒
3. **section3.mp3** - 自然意象与内心平静: ~90秒
4. **section4.mp3** - 愿景设想的心理准备: ~80秒
5. **section5.mp3** - 结束与回归: ~60秒

## 🎙️ 录音建议

### 录制环境
- 安静的空间
- 关闭所有噪音源
- 使用防喷罩

### 音频处理
- 去除背景噪音
- 标准化音量
- 添加淡入淡出
- 导出为MP3格式 (320kbps)

### 女声特点
- 温暖、柔和
- 语速缓慢 (~100-120字/分钟)
- 音调平稳,不起伏太大
- 停顿时保持自然呼吸声

## ✅ 系统会自动

1. **检测音频文件** - 如果存在则播放,不存在则跳过
2. **同步文字与语音** - 文字引导词与语音同步显示
3. **自动切换阶段** - 语音播放完毕自动进入下一阶段
4. **音量控制** - 背景音乐降低,语音清晰可听
